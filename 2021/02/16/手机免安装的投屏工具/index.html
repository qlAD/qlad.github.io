手机免安装的投屏工具

Created2021-02-16|Updated2021-06-19|安卓
|Word count:358|Reading time:1min|Post View:

1.工具介绍 Scrcpy

这个应用程序提供通过 USB (或 TCP/IP )连接的 Android 设备的显示和控制。它不需要任何 root 权限。它可以在 GNU/Linux 、Windows 和 macOS 上工作。

2.下载工具

访问 GitHub 项目 scrcpy

在它的介绍文件往下翻可以看到 Get the app

我的电脑是Windows，并且提供了多种方式，这里我选择下载 zip 文件

到此，工具就准备好了

注意：
需要投屏的 Android 设备至少需要 Android 5.0。

3.投屏

有线投屏

打开手机的 adb 调试功能（必需）

用数据线将电脑和手机连接
双击文件夹中的 scrcpy.exe 就能将手机屏幕映射到电脑上

无线投屏

打开手机的网络 adb 调试功能（必需）

打开 Windows 的 Powershell


进入到 scrcpy 的目录

用目录中的 adb 工具连接手机

注意：
IP和端口是手机分配的，要与之对应

用命令行打开 scrcpy 实现无线投屏，最后放一张实际连接图**
